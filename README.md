## About The Project

Data Science is changing very quickly. In this repository I explore different ideas from fundamental to cutting edge. My goal is to make the code clean and understandable to myself. 

Although it'd be nice if my project can help others understand concepts, that is not the intention of this repository. 

The main reason I share this is because friends may be interested in what I'm working on, and provide a way to easily share inner workings of things.
